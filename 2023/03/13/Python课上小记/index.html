过7游戏：

Question:

  • 过七游戏：输入一个整数，如果这个整数的下一个数是7的倍数或者包含“7”则输出“过”，否则输出这个数。

Answer:

num = int(input('输入数字：')) + 1
print('需要输入正整数') if num < 2 else print('过') if '7' in str(num) or num % 7 == 0 else print(num)

Analysis:

  1. 注意题目要求：需要判断的数是输入的下一个数
  2. line1: 将用户输入的数字加一并存入num
  3. line2: 代码等同于：
if num < 2:  # 判断自增后的数是否小于2，只有大于1的整数才是正整数
   print('需要输入正整数')
else:
   if '7' in str(num) or num % 7 == 0:  # 判断"7"是否在num内或者是否能整除7
       print('过')
   else:
       print(num)

猜大小：

Question:

  • 程序生成一个随机数，让用户猜这个数是多少，如果输入的数小于随机数则返回”猜低了”，反之返回”猜高了”，并让用户接着输入下一个数，直到猜对返回这个随机数和共计的次数。

Answer:

import random
ans, count = random.randint(0, 100), 1  # 生成随机数并初始化次数
# 使用海象运算符*对每次输入的价格进行赋值的同时判断是否等于随机数
while (num := int(input('请输入价格：'))) != ans:
   count += 1  # 不等于则次数+1
   # 如果输入的数大于随机数则输出"猜高了"，反之输出"猜低了"
   print('猜高了' if num > ans else '猜低了')
print(f'恭喜猜对！答案是{num}，一共猜了{count}次。')

Analysis:

  1. 海象运算符

质数判定：

Question:

  • ⽤户输⼊⼀个⼤于1的正整数 n ，判断这个数是不是质数
    如果⼀个⼤于1的正整数只能被1和⾃⼰整除，则这个数为质数

Answer:

num = int(input('输入：'))
print(f"输出：{'不是' if any(num % i == 0 for i in range(2, int(num ** 0.5 + 1))) else '是'}质数")

Analysis:

  1. int(num ** 0.5 + 1)
    获取n开平方后的向上取整
  2. any(num % i == 0 for i in range(2, int(num ** 0.5 + 1)))
    判断num是否能被2到第一步的数中的数字整除，若能被整除则不是质数，反之则是质数
    any函数用于判断序列中是否为非空

⽇期计算：

Question:

  • ⽤户输⼊⼀个正整数 n ，该数字代表这是2023年的第n天，输出这⼀天是⼏⽉⼏号。
    思考：如何同时输出这⼀天是星期⼏？

Answer:

default = num = int(input('输入：'))  # 初始化输入日期，其中default给星期几使用
time_lst, month = [31, 28, 31, 30, 31, 30, 31, 31, 30, 31, 30, 31], 0  # 2023年时间列表
while (num := num - time_lst[month]) > 0:
   month += 1
print(f"输出：{month + 1}月{num + time_lst[month]}日")
print(f"输出：星期{['日', '一', '二', '三', '四', '五', '六'][default % 7 - 1]}")

Analysis:

  1. while (num := num - time_lst[month]) > 0:
    time_lst[month]为month+1这个月的时间，使用海象运算符让num循环减去下个月的天数，如果num减去天数后小于0则代表剩余天数不够下个月的天数，则跳出循环
  2. {num + time_lst[month]}日
    因为num还循环体中已经被减去了最后一个月的天数，所以输出的时候要加上最后一个月的天数。
  3. 星期{[‘日’, ‘一’, ‘二’, ‘三’, ‘四’, ‘五’, ‘六’][default % 7 - 1]}
    利用初始化的default模除7再减去1就得到星期几，因为2023-01-01这天是周日

判断标识符是否合法：

Question:

  • ⽤户输⼊⼀个字符串，判断这个字符串是否是合法的？

Answer:

import keyword
user_input = input('输入：')
print(f"输出：{'非法' if not user_input.isidentifier() or keyword.iskeyword(user_input) else '合法'}")

Analysis:

  1. user_input.isidentifier()
    python内置函数isidentifier()，用于返回字符串是否合法，若user_input能构成变量名(关键字除外)，则返回True，反之返回False；用not取反，若不合法则最终返回为True输出”非法”
  2. keyword.iskeyword(user_input)
    用于返回user_input是否为关键字，因为第一步的函数无法判断关键字，故用此函数判断，若是关键字则返回True，反之返回False，不合法则输出”非法”

字符串切片I：

Question:

  • 用户输入一个字符串，输出字符串的前6个字符，如果长度不足6，则输出全部字符。

Answer:

print(input('请输入字符串：')[:6])

Analysis:

  1. 直接对输入字符串进行切片，切片用法为[start：end：step]
    省去start则默认为0，省去step则默认为1
    切片不会越界，所以无需考虑长度不足6的情况，直接切片即可

s = input('请输入字符串：')

字符串查找：

Question:

  • 众所周知，Python字符串的find⽅法只能找到第⼀个⼦串的下标，请编写代码，完成以下功能：
  • ⽤户输⼊两个字符串 sent 和 sub ，如果 sent 字符串中包含 sub ，则打印出所有的下标，否则打印 -1

Answer:

sent, sub = input('请输入字符串：'), input('请输入查找字串：')
print('\n'.join([str(i) for i in range(len(sent)) if sent[i] == sub]) or '-1')

Analysis:

  1. [i for i in range(len(sent)) if sent[i] == sub]
    利用for循环判断sent中是否有等于sub的，如果有则将下标存入这个列表中
  2. map(str, [i for i in range(len(sent)) if sent[i] == sub])
    利用map函数将上一步列表里的int转为字符串，用于后续join函数输出
  3. ‘\n’.join()
    将上一步的列表进行拼接，利用\n实现每一行一个下标
  4. print(‘\n’.join() or -1)
    若or左侧，也就是第三步中没有结果则判定为False，输出-1

跑⻢灯：

Question:

  • 制作跑马灯并实现以下效果：
    【欢迎XXX来到深圳职业技术学院】

Answer:

import time
sent = "欢迎XXX来到深圳职业技术学院"
while True:
   print(f"\r【{sent}】", end="") # 利用\r实现每次输出之前都把上次输出的内容清空
   time.sleep(0.5)
   sent = sent[1:] + sent[0]  # 生成新的字符串供下次输出使用

邮箱地址合法性判断:

Question:

  • ⽤户输⼊⼀个字符串，判断该字符串是否是合法的邮箱地址，合法的邮箱地址符合以下规则：
    电⼦邮件地址格式为 域内部分@域名 ，域内部分需要符合以下规则：
  1. 最⻓64个字符
  2. 可以由英⽂⼩写字⺟ a 到 z 和⼤写字⺟ A 到 Z 、数字 0 到 9 、英⽂句号 . 、短横线 - 或者下划线 _ 组成
  3. 英⽂句号 . 、短横线 - 或者下划线 _ 不能作为⾸尾字符，也不能连续出现\
  • 域名部分除了以上规则，还需要符合以下规则：
  1. 不能是纯数字
  2. 必须包含英⽂句号 . ，但不能是⾸尾字符，也不能连续出现

Answer:

def is_email(email_in):
   def is_local_true(is_local):
       # 判断长度是否大于64、判断是否有连续的". - _"
       if len(is_local) > 64 or '..' in is_local or '--' in is_local or '__' in is_local:
           return False
       if is_local[0] in ['.', '-', '_'] or is_local[-1] in ['.', '-', '_']:  # 判断首位是否为". - _"
           return False
       # 判断是否有除字母数字和". - _"之外的字符
       for in_local in is_local:
           if in_local not in ['.', '-', '_'] and not in_local.isalnum():
               return False
       return True
   if email_in.count('@') != 1 or email_in.startswith('@') or email_in.endswith('@'):  # 判断邮箱基本格式是否合法
       return False
   else:
       local, domain = email_in.split('@')  # 使⽤split⽅法分割，local是域内部分，domain是域名
       # 判断域内部分是否合法、判断域名部分是否合法、判断域名部分是否存在'.'、判断域名部分是否为纯数字
       if not is_local_true(local) or not is_local_true(domain) or '.' not in domain or domain.isdigit():
           return False
       return True
email = input("输⼊邮箱地址：").strip()
print(f"{email}是{'有效' if is_email(email) else '无效'}的邮箱地址")

IPv4地址合法性判断：

Question:

  • ⽤户输⼊⼀个字符串，该字符串只由数字 0 到 9 和英⽂句号 . 组成判断该字符串是否为合法的IPv4地址。

Answer:

inp = input('请输入IPv4地址：')
print('非法IP地址' if inp.count('.') != 3 or sum([1 for i in inp.split('.') if i.isnumeric() and 0 <= int(i) < 256]) != 4 else '合法IP地址')

Analysis:

  1. inp.count(‘.’) != 3
    判断IP地址是否只包含三个’.’，用or参数拼接，若前半部分为False则不判断后半部分
  2. [1 for i in inp.split(‘.’) if i.isnumeric() and 0 <= int(i) < 256]
    利用split进行分割，使用for进行遍历，若不为空且大于等于0小于256则在列表中添加’1’
  3. sum([1 for i in inp.split(‘.’) if i.isnumeric() and 0 <= int(i) < 256]) != 4
    利用sum函数对第二步中的列表进行求和，若等于4则点分十进制的4个部分都合法
