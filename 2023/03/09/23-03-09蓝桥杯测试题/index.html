请找到⼀个⼤于 2022 的最⼩数，这个数转换成⼗六进制之后，所有的数位（不含前导 0）都为字⺟（A 到 F）。
请将这个数的⼗进制形式作为答案提交。

def fun1():
    cur = 2022
    while True:
        for i in hex(cur)[2:]:
            if i.isnumeric():
                break
        else:
            return cur
        cur += 1

在 Excel 中，列的名称使⽤英⽂字⺟的组合。前 26 列⽤⼀个字⺟，依次为 A 到 Z，接下来 26*26 列使⽤两个字⺟的组合，依次为 AA 到 ZZ。
请问第 2022 列的名称是什么？

def fun2():
    cur, ans = 2022, ''
    while True:
        if cur % 26 == 0:
            ans += 'Z'
            cur = cur // 26 - 1
        else:
            ans += chr(ord('A') + cur % 26 - 1)
            cur //= 26
        if cur == 0:
            return ans[::-1]

对于⼀个⽇期，我们可以计算出年份的各个数位上的数字之和，也可以分别计算⽉和⽇的各位数字之和。请问从1900 年 1 ⽉ 1 ⽇⾄ 9999 年 12 ⽉ 31 ⽇，总共有多少天，年份的数位数字之和等于⽉的数位数字之和加⽇的数位数字之和。
例如，2022年11⽉13⽇满⾜要求，因为 2+0+2+2=(1+1)+(1+3) 。
请提交满⾜条件的⽇期的总数量。

def fun3():
    import datetime
    ans = 0
    for i in range(1900, 9999):
        for j in range(1, 13):
            for k in range(1, 32):
                if sum(map(int, str(i))) == j + k:
                    try:
                        datetime.datetime(i, j, k)
                        ans += 1
                    except ValueError:
                        pass
    return ans

⼩蓝负责⼀块区域的信号塔安装，整块区域是⼀个⻓⽅形区域，建⽴坐标轴后，⻄南⻆坐标为 (0, 0)， 东南⻆坐标为 (W, 0)， ⻄北⻆坐标为 (0, H)， 东北⻆坐标为 (W, H)。其中 W, H 都是整数。
他在 n 个位置设置了信号塔，每个信号塔可以覆盖以⾃⼰为圆⼼，半径为 R 的圆形（包括边缘）。
为了对信号覆盖的情况进⾏检查，⼩蓝打算在区域内的所有横纵坐标为整数的点进⾏测试，检查信号状态。其中横坐标范围为 0 到 W，纵坐标范围为 0 到 H，总共测试 (W+1) (H+1) 个点。
给定信号塔的位置，请问这 (W+1)(H+1) 个点中有多少个点被信号覆盖。
输⼊格式：
输⼊第⼀⾏包含四个整数 W, H, n, R，相邻整数之间使⽤⼀个空格分隔。
接下来 n ⾏，每⾏包含两个整数 x, y，表示⼀个信号塔的坐标。信号塔可能重合，表示两个信号发射器装在了同⼀个位置。

def fun7():
    info = list(map(int, input().split()))
    lst, ans = [], 0
    for i in range(info[2]):
        lst.append(list(map(int, input().split())))
    for i in range(info[0]):
        for j in range(info[1]):
            for k in lst:
                if ((i - k[0]) ** 2 + (j - k[1]) ** 2) ** 0.5 <= info[3]:
                    ans += 1
                    break
    return ans

⼩蓝有⼀个 n * m ⼤⼩的矩形⽔域，⼩蓝将这个⽔域划分为 n ⾏ m 列，⾏数从 1 到 n 标号，列数从 1 到 m 标号。每⾏和每列的宽度都是单位 1 。
现在，这个⽔域⻓满了⽔草，⼩蓝要清理⽔草。
每次，⼩蓝可以清理⼀块矩形的区域，从第 r1 ⾏（含）到第 r2 ⾏（含）的第 c1 列（含）到 c2 列（含）。
经过⼀段时间清理后，请问还有多少地⽅没有被清理过。
输⼊格式：
输⼊第⼀⾏包含两个整数 n, m，⽤⼀个空格分隔。
第⼆⾏包含⼀个整数 t ，表示清理的次数。
接下来 t ⾏，每⾏四个整数 r1, c1, r2, c2，相邻整数之间⽤⼀个空格分隔，表示⼀次清理。请注意输⼊的顺序。

def fun8():
    n, m = map(int, input().split())
    jz = [[0 for _ in range(m)] for _ in range(n)]
    for i in range(int(input())):
        a, b, c, d = map(int, input().split())
        for j in range(a - 1, c):
            for k in range(b - 1, d):
                jz[j][k] = 1
    ans = 0
    for i in jz:
        ans += (m - sum(i))
    return ans

⼩蓝准备在⼀个空旷的场地⾥⾯滑⾏，这个场地的⾼度不⼀，⼩蓝⽤⼀个 n ⾏ m 列的矩阵来表示场地，矩阵中的数值表示场地的⾼度。
如果⼩蓝在某个位置，⽽他上、下、左、右中有⼀个位置的⾼度（严格）低于当前的⾼度，⼩蓝就可以滑过去，滑动距离为 1 。
如果⼩蓝在某个位置，⽽他上、下、左、右中所有位置的⾼度都⼤于等于当前的⾼度，⼩蓝的滑⾏就结束了。
⼩蓝不能滑出矩阵所表示的场地。
⼩蓝可以任意选择⼀个位置开始滑⾏，请问⼩蓝最多能滑⾏多远距离。
输⼊格式：
输⼊第⼀⾏包含两个整数 n, m，⽤⼀个空格分隔。
接下来 n ⾏，每⾏包含 m 个整数，相邻整数之间⽤⼀个空格分隔，依次表示每个位置的⾼度。

def fun9():
    def dfs(i, j):
        max_num = 0
        for x, y in [(i - 1, j), (i + 1, j), (i, j - 1), (i, j + 1)]:
            if 0 <= x <= n - 1 and 0 <= y <= m - 1 and matrix[x][y] < matrix[i][j] and not visited[x][y]:
                visited[i][j] = True
                max_num = max(dfs(x, y), max_num)
                visited[i][j] = False
        return max_num + 1
    n, m = map(int, input().split())
    ans, matrix, visited = 0, [], [[False] * m for _ in range(n)]
    for _ in range(n):
        matrix.append(list(map(int, input().split())))
    for a in range(n):
        for b in range(m):
            visited[a][b] = True
            ans = max(dfs(a, b), ans)
            visited[a][b] = False
    return ans

⼩蓝有⼀个序列a[1], a[2], …, a[n]。
给定⼀个正整数 k，请问对于每⼀个 1 到 n 之间的序号 i， a[i−k],a[i−k+1],…,a[i+k] 这 2k+1 个数中的最⼩值是多少？
当某个下标超过 1 到 n 的范围时，数不存在，求最⼩值时只取存在的那些值。
输⼊格式：
输⼊的第⼀⾏包含⼀整数 n。
第⼆⾏包含 n 个整数，分别表示 a[1], a[2], …, a[n]。
第三⾏包含⼀个整数 k 。

def fun10():
    _, lst, k, ans = input(), input().split(), input(), []
    for i in range(len(lst)):
        ans.append(min(list(map(int, lst[max(i - int(k), 0):i + int(k) + 1]))))
    return ans
