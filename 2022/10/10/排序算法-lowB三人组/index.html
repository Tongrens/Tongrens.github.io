排序算法-lowB三人组

2022/10/10 笔记 python sort

冒泡排序

def bubble_sort(lst):
    for i in range(len(lst) - 1):
        flag = False
        for j in range(len(lst) - i - 1):
            if lst[j] > lst[j + 1]:
                lst[j], lst[j + 1] = lst[j + 1], lst[j]
                flag = True
        if not flag:
            return

li = [4, 6, 2, 3, 1, 5, 7, 9, 8]
bubble_sort(li)
print(li)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]

若无序区的相邻两个数前面的数大于后面的则交换
(优化)每一趟结束后判断flag，若无修改则直接退出

i为第几趟(0开始)
j为遍历无序区的下标

有序区为0——(i-1)
无序区为i——(len(lst)-1)

时间复杂度：O(n²)
空间复杂度：O(1)
稳定性：稳定

插入排序

def insert_sort(lst):
    for i in range(1, len(lst)):
        tmp = lst[i]
        j = i-1
        while j >= 0 and lst[j] > tmp:
            lst[j+1] = lst[j]
            j -= 1
        lst[j+1] = tmp

li = [4, 6, 2, 3, 1, 5, 7, 9, 8]
insert_sort(li)
print(li)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]

初始手里一张牌(lst[0])
每次摸到一张牌(lst[i])，判断i前面的牌(j)是否比摸到的牌大，大则将前面的牌向后移一位，否则j向前移动一位并进行下次判断
若lst[j]<lst[i]则将摸到的牌的值插入到lst[j+1]

i为无序区第一个数
j为有序区为以-1步进遍历的下标

有序区为0——(i-1)
无序区为i——(len(lst)-1)

时间复杂度：O(n²)
空间复杂度：O(1)
稳定性：稳定

选择排序

def select_sort(lst):
    for i in range(len(lst)-1):
        tmp = i
        for j in range(i, len(lst)):
            if lst[j] < lst[tmp]:
                tmp = j
        lst[i], lst[tmp] = lst[tmp], lst[i]

li = [4, 6, 2, 3, 1, 5, 7, 9, 8]
select_sort(li)
print(li)  # [1, 2, 3, 4, 5, 6, 7, 8, 9]

每一趟找出无序区的最小数并与无序区第一个数交换

i为第几趟(0开始)
j为遍历无序区的下标

有序区为0——(i-1)
无序区为i——(len(lst)-1)

时间复杂度：O(n²)
空间复杂度：O(1)
稳定性：不稳定
