基本VLAN特性配置与管理

VLAN基础

VLAN（虚拟局域网）是一个将物理LAN逻辑划分为若干个虚拟的LAN，目的是为了缩小广播域，使广播域进行二层（数据链路层）通信，不同VLAN间的主机是不能互相通信的，必须经过第三层（网络层）进行通信。
VLAN既能限制广播网络，又可增强局域网的安全性，还提高了网络的健壮性。
在同一个物理交换机上不可能存在两个相同的VLAN，在不同交换机上则可以有相同VLAN

VLAN帧

VLAN Tag:标识所属VLAN，遵循IEEE 802.1Q标注，支持802.1Q协议的交换机既可以发送有标记帧也可以发送无标记帧（有标记帧加入了4字节的801.1Q Tag，反之则没有），交换机内部的数据包一律携带Tag。

# 链表
class Node:
    def __init__(self, item=None):
        self.item = item
        self.next = None
        self.prior = Node  # 双链表

# 头插法
def create_linklist_head(lst):
    head = Node(lst[0])
    for element in lst[1:]:
        node = Node(element)
        head.prior = node  # 双链表
        node.next = head
        head = node
    return head

# 前序遍历  E,A,C,B,D,G,F,
def pre_order(root):
    if root:
        print(root.data, end=',')
        pre_order(root.lchild)
        pre_order(root.rchild)

# 中序遍历  A,B,C,D,E,G,F,
def in_order(root):
    if root:
        in_order(root.lchild)
        print(root.data, end=',')
        in_order(root.rchild)
