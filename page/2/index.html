基本VLAN特性配置与管理

VLAN基础

VLAN（虚拟局域网）是一个将物理LAN逻辑划分为若干个虚拟的LAN，目的是为了缩小广播域，使广播域进行二层（数据链路层）通信，不同VLAN间的主机是不能互相通信的，必须经过第三层（网络层）进行通信。
VLAN既能限制广播网络，又可增强局域网的安全性，还提高了网络的健壮性。
在同一个物理交换机上不可能存在两个相同的VLAN，在不同交换机上则可以有相同VLAN

VLAN帧

VLAN Tag:标识所属VLAN，遵循IEEE 802.1Q标注，支持802.1Q协议的交换机既可以发送有标记帧也可以发送无标记帧（有标记帧加入了4字节的801.1Q Tag，反之则没有），交换机内部的数据包一律携带Tag。

802.1Q Tag各字段含义：

TPID：长度2bytes，表示帧类型，取值为0x8100为802.1Q帧，如果不支持的设备接受到则丢弃
PRI：长度3bits，表示帧优先级，取值范围0-7，越大越优，用于QOS
CFI：长度1bits，表示MAC地址是否为经典格式，0是经典1是非经典，在以太网中值为1
VID：长度12bits，表示VLAN ID

VLAN特性

华为交换机支持5种VLAN划分：基于端口划分、基于子网划分、基于MAC地址划分、基于协议划分、基于策略划分；其中端口划分属于静态VLAN划分，其余的属于动态VLAN划分
（除了端口划分其余都要在Hybrid口上划分）

常见的VLAN间通信有三层VLANIF接口、三层以太网子接口和VLAN switch方案

VLAN聚合：是指Super VLAN技术，他是在一个主VLAN下包括多个同IP网段的VLAN技术，但只需要为主VLANIF配置IP地址，各从VLAN中的用户主机就可以通过主VLAN的VLANIF接口作为缺省网关进行通信

VLAN映射：在一些交换机上不支持全范围（1-4094）的VLAN ID，而只支持1-512的VLAN ID就与公网的VLAN ID实现冲突；于是就产生了一种可以实现在用户VLAN ID与运营商VLAN ID进行相互转换的技术。

语音VLAN：（Voice VLAN）是相对传统VLAN数据而言的，他是针对不同类型的用户语言流进行划分的，并能自动修改数据帧的优先级，以提高语音数据的传输质量

VLAN透传：可以使交换机直接透明传输指定VLAN的数据帧，不上送CPU处理，也不用去识别VLAN标签，提高了转发效率

QINQ协议：是基于IEEE 802.1Q技术的一种二层隧道协议。在公网中传输的数据一般分为公网VLAN标签和私网VLAN标签，QINQ的核心思想是为了将私网标签封装在公网标签中，为用户提供一种较为简单的二层VPN隧道

二层以太网端口

二层以太网端口类型有：access、trunk、hybrid和QINQ四种端口类型

Access端口：主要用来连接用户主机的二层以太网端口，仅允许一个VLAN通过，发送的以太网帧永远是不带标签的（Untagged）
Trunk端口：主要用来连接与其他交换机的二层端口，允许多个VLAN通过，发送的帧都是带标签的
Hybrid端口：Hybrid端口可以说是access端口和trunk端口的混合体，是一种特殊的二层以太网端口，既可以连接主机也可以连接交换机，可以允许一个或多个VLAN通过，并可以选择带标签或不带标签。
QINQ端口：QINQ端口是专用于QINQ协议的二层以太网端口，可以给数据帧加上双层VLAN标签，从而可以支持多达4094*4094个VLAN。

二层以太网链路

以太网链路分为接入链路（Access link）和干道链路（Trunk link）

接入链路：是交换机直接连接用户主机的链路。通常情况下主机并不需要识别自己是属于哪个VLAN，所以接入链路发送和接受的数据帧都是不带标签（Untagged）的，但接入链路不一定只允许一个VLAN通过，只是access端口才仅允许一个VLAN通过，当属于Hybrid端口时同样是允许多个VLAN通过的

干道链路：是交换机与交换机互联的链路。可以承载多个不同VLAN数据传输，干道链路两端的设备需要识别VLAN标签，所以在干道链路的数据都是带标签（Tagged）的，除了该链路的PVID所属VLAN（默认VLAN1）

VLAN间通信

  1. 三层VLANIF接口方案（VLANIF接口作为对应VLAN内的用户缺省网关）
  2. 三层以太网子接口方案（同时具备三层以太网物理接口和二层以太网物理接口的双重特性）
  3. VLAN Switch方案（一种按照VLAN标签进行数据转发的技术，需要预先在各交换机上配置静态转发路径，符合条件即可转发到对应的出接口，无需查看MAC地址，提高转发效率和安全性，能避免MAC地址攻击和广播风暴）

# 链表
class Node:
    def __init__(self, item=None):
        self.item = item
        self.next = None
        self.prior = Node  # 双链表

# 头插法
def create_linklist_head(lst):
    head = Node(lst[0])
    for element in lst[1:]:
        node = Node(element)
        head.prior = node  # 双链表
        node.next = head
        head = node
    return head

# 尾插法
def create_linklist_tail(lst):
    head = Node(lst[0])
    tail = head
    for element in lst[1:]:
        node = Node(element)
        tail.next = node
        tmp = tail
        tail = node
        tail.prior = tmp  # 26&28双链表
    return head

# 输出链表
def print_linklist(lk):
    while lk:
        print(lk.item, end=',')
        lk = lk.next
    print()

# 头插法eg
lk1 = create_linklist_head([1, 2, 3])
print(lk1.item)  # 3
print(lk1.next.item)  # 2
print(lk1.next.prior.item)  # 3(双链表)
print_linklist(lk1)  # 3,2,1,
# 尾插法eg
lk2 = create_linklist_tail([1, 2, 3])
print(lk2.item)  # 1
print(lk2.next.item)  # 2
print(lk2.next.prior.item)  # 1(双链表)
print_linklist(lk2)  # 1,2,3,
# 插入元素
p = Node(5)
p.next = lk2.next
lk2.next.prior = p
p.prior = lk2   # 54&55双链表
lk2.next = p
print_linklist(lk2)  # 1,5,2,3,
# 删除元素
lk2.next.next.prior = lk2
lk2.next = lk2.next.next
print_linklist(lk2)  # 1,2,3,
"""

按元素查找：   O(n)      O(n)
按下标查找：   O(1)      O(n)
插入&删除：   O(n)      O(1)
"""

# AVLTree
# https://www.bilibili.com/video/BV1uA411N7c5?p=74&vd_source=2493c37afb3b31338e613a5ec70a3fcd
from BST import BiTreeNode, BST

class AVLNode(BiTreeNode):
    def __init__(self, data):
        BiTreeNode.__init__(self, data)
        self.bf = 0

class AVLTree(BST):
    def __init__(self, li=None):
        BST.__init__(self, li)

    def rotate_left(self, p, c):
        lc = c.lchild
        p.rchild = lc
        if lc:
            lc.parent = p
        c.lchild = p
        p.parent = c
        p.bf = 0
        c.bf = 0
        return c

    def rotate_right(self, p, c):
        rc = c.rchild
        p.lchild = rc
        if rc:
            rc.parent = p
        c.rchild = p
        p.parent = c
        p.bf = 0
        c.bf = 0
        return c

    def rotate_right_left(self, p, c):
        g = c.lchild
        rrc = g.rchild
        c.lchild = rrc
        if rrc:
            rrc.parent = c
        g.rchild = c
        c.parent = g
        rlc = g.lchild
        p.rchild = rlc
        if rlc:
            rlc.parent = p
        g.lchild = p
        p.parent = g
        if g.bf > 0:
            p.bf = -1
            c.bf = 0
        elif g.bf < 0:
            p.bf = 0
            c.bf = 1
        else:
            p.bf = 0
            c.bf = 0
        return g

    def rotate_left_right(self, p, c):
        g = c.rchild
        rlc = g.lchild
        c.rchild = rlc
        if rlc:
            rlc.parent = c
        g.lchild = c
        c.parent = g
        rrc = g.rchild
        p.lchild = rrc
        if rrc:
            rrc.parent = p
        g.rchild = p
        p.parent = g
        if g.bf > 0:
            p.bf = 0
            c.bf = -1
        elif g.bf < 0:
            p.bf = 1
            c.bf = 0
        else:
            p.bf = 0
            c.bf = 0
        return g

    def insert(self, val):
        p = self.root
        if not p:
            self.root = AVLNode(val)
            return
        while True:
            if val < p.data:
                if p.lchild:
                    p = p.lchild
                else:
                    p.lchild = AVLNode(val)
                    p.lchild.parent = p
                    node = p.lchild
                    break
            elif val > p.data:
                if p.rchild:
                    p = p.rchild
                else:
                    p.rchild = AVLNode(val)
                    p.rchild.parent = p
                    node = p.rchild
                    break
            else:
                return
        while node.parent:
            g = node.parent.parent
            x = node.parent
            if node == node.parent.lchild:
                if node.parent.bf < 0:
                    if node.bf > 0:
                        n = self.rotate_left_right(node.parent, node)
                    else:
                        n = self.rotate_right(node.parent, node)
                elif node.parent.bf == 0:
                    node.parent.bf = -1
                    node = node.parent
                    continue
                else:
                    node.parent.bf = 0
                    break
            else:
                if node.parent.bf > 0:
                    if node.bf < 0:
                        n = self.rotate_right_left(node.parent, node)
                    else:
                        n = self.rotate_left(node.parent, node)
                elif node.parent.bf == 0:
                    node.parent.bf = 1
                    node = node.parent
                    continue
                else:
                    node.parent.bf = 0
                    break
            n.parent = g
            if g:
                if x == g.lchild:
                    g.lchild = n
                else:
                    g.rchild = n
                break
            else:
                self.root = n
                break

# BST
class BiTreeNode:
    def __init__(self, data):
        self.data = data
        self.lchild = None
        self.rchild = None
        self.parent = None

class BST:
    def __init__(self, lst=None):
        self.root = None
        if lst:
            for val in lst:
                self.insert(val)

    def insert(self, val):
        p = self.root
        if not p:
            self.root = BiTreeNode(val)
            return
        while True:
            if val < p.data:
                if p.lchild:
                    p = p.lchild
                else:
                    p.lchild = BiTreeNode(val)
                    p.lchild.parent = p
                    return
            elif val > p.data:
                if p.rchild:
                    p = p.rchild
                else:
                    p.rchild = BiTreeNode(val)
                    p.rchild.parent = p
                    return
            else:
                return

    def find(self, val):
        p = self.root
        while p:
            if val > p.data:
                p = p.rchild
            elif val < p.data:
                p = p.lchild
            else:
                return p
        return None

    def __delete_node_1(self, node):  # 删除叶子节点
        if not node.parent:     # 只有一个根节点
            self.root = None
        elif node == node.parent.lchild:
            node.parent.lchild = None
        else:
            node.parent.rchild = None

    def __delete_node_2_left(self, node):  # 删除非叶子节点，只有左子树
        if not node.parent:
            self.root = node.lchild
            node.lchild.parent = None
        elif node == node.parent.lchild:
            node.parent.lchild = node.lchild
            node.lchild.parent = node.parent
        else:
            node.parent.rchild = node.lchild
            node.lchild.parent = node.parent

    def __delete_node_2_right(self, node):  # 删除非叶子节点，只有右子树
        if not node.parent:
            self.root = node.rchild
            node.rchild.parent = None
        elif node == node.parent.lchild:
            node.parent.lchild = node.rchild
            node.rchild.parent = node.parent
        else:
            node.parent.rchild = node.rchild
            node.rchild.parent = node.parent

    def delete(self, val):
        if self.root:
            node = self.find(val)
            if not node:
                raise ValueError('\"' + str(val) + '\" not in tree')
            if not node.lchild and not node.rchild:
                self.__delete_node_1(node)
            elif not node.rchild:
                self.__delete_node_2_left(node)
            elif not node.lchild:
                self.__delete_node_2_right(node)
            else:
                min_node = node.rchild
                while min_node.lchild:
                    min_node = min_node.lchild
                node.data = min_node.data
                if min_node.rchild:
                    self.__delete_node_2_right(min_node)
                else:
                    self.__delete_node_1(min_node)

    def order(self, root):
        if root:
            self.order(root.lchild)
            print(root.data, end=',')
            self.order(root.rchild)

# BiTreeNode
class BiTreeNode:
    def __init__(self, data):
        self.data = data
        self.lchild = None
        self.rchild = None

# 前序遍历  E,A,C,B,D,G,F,
def pre_order(root):
    if root:
        print(root.data, end=',')
        pre_order(root.lchild)
        pre_order(root.rchild)

# 中序遍历  A,B,C,D,E,G,F,
def in_order(root):
    if root:
        in_order(root.lchild)
        print(root.data, end=',')
        in_order(root.rchild)

# 后序遍历  B,D,C,A,F,G,E,
def post_order(root):
    if root:
        post_order(root.lchild)
        post_order(root.rchild)
        print(root.data, end=',')

# 层次遍历  E,A,G,C,F,B,D,
def level_order(root):
    from collections import deque
    queue = deque()
    queue.append(root)
    while len(queue) > 0:
        node = queue.popleft()
        print(node.data, end=',')
        if node.lchild:
            queue.append(node.lchild)
        if node.rchild:
            queue.append(node.rchild)
