过7游戏：

Question:

  • 过七游戏：输入一个整数，如果这个整数的下一个数是7的倍数或者包含“7”则输出“过”，否则输出这个数。

Answer:

num = int(input('输入数字：')) + 1
print('需要输入正整数') if num < 2 else print('过') if '7' in str(num) or num % 7 == 0 else print(num)

Analysis:

  1. 注意题目要求：需要判断的数是输入的下一个数
  2. line1: 将用户输入的数字加一并存入num
  3. line2: 代码等同于：
if num < 2:  # 判断自增后的数是否小于2，只有大于1的整数才是正整数
   print('需要输入正整数')
else:
   if '7' in str(num) or num % 7 == 0:  # 判断"7"是否在num内或者是否能整除7
       print('过')
   else:
       print(num)

请找到⼀个⼤于 2022 的最⼩数，这个数转换成⼗六进制之后，所有的数位（不含前导 0）都为字⺟（A 到 F）。
请将这个数的⼗进制形式作为答案提交。

def fun1():
    cur = 2022
    while True:
        for i in hex(cur)[2:]:
            if i.isnumeric():
                break
        else:
            return cur
        cur += 1
