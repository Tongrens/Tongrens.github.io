接口及以太网链路配置及管理

以太网接口属性

二层接口是一种物理接口，属于数据链路层，不能配置IP地址，可以对接收到的报文经行二层转发。

三层子接口是一种逻辑接口，属于网络层，可以配置IP地址，主要用来实现收发VLAN报文。可以通过配置使一个二层接口绑定多个三层子接口，从而使得不同VLAN报文从不同子接口转发。

接口类型

接口类型分为管理接口、物理接口和逻辑接口
管理接口用于登陆到设备经行配置与管理，此接口不承担业务传输
物理接口是真实存在的可承担业务传输的接口，一般由电口与光口组成
逻辑接口是指能实现数据交换但物理上不存在的接口

端口隔离

端口隔离分为单向隔离和端口隔离组，且都仅适用于同意交换机上不同端口间的隔离，且一个端口可以加入多个端口隔离组

  1. 单向隔离指禁止某个端口发送的报文到达其他端口，而不禁止其他端口的报文到达本端口

  2. 而端口隔离组内的接口则是互相隔离的

基本VLAN特性配置与管理

VLAN基础

VLAN（虚拟局域网）是一个将物理LAN逻辑划分为若干个虚拟的LAN，目的是为了缩小广播域，使广播域进行二层（数据链路层）通信，不同VLAN间的主机是不能互相通信的，必须经过第三层（网络层）进行通信。
VLAN既能限制广播网络，又可增强局域网的安全性，还提高了网络的健壮性。
在同一个物理交换机上不可能存在两个相同的VLAN，在不同交换机上则可以有相同VLAN

VLAN帧

VLAN Tag:标识所属VLAN，遵循IEEE 802.1Q标注，支持802.1Q协议的交换机既可以发送有标记帧也可以发送无标记帧（有标记帧加入了4字节的801.1Q Tag，反之则没有），交换机内部的数据包一律携带Tag。

# 链表
class Node:
    def __init__(self, item=None):
        self.item = item
        self.next = None
        self.prior = Node  # 双链表

# 头插法
def create_linklist_head(lst):
    head = Node(lst[0])
    for element in lst[1:]:
        node = Node(element)
        head.prior = node  # 双链表
        node.next = head
        head = node
    return head
